<br>
`14、Exercise1 二叉树删除，思路：删除节点无子节点和只有一个节点，这两种情况都是比较简单的，
第三种情况，删除节点既有左节点又有右节点，那么从该节点的子树集合中找到大于该节点的最小的一个，来代替被删除的节点，那就要在这
节点右子树上一直往左找。这个节点比被删除的节点的右节点小，且比左节点大，那这个节点就叫做被删除节点的后继节点，用这个节点来代替被删除节点
，再删除后继节点`
